# POS System for Fast-Food Restaurant

# Feature 

* Administrator Side
* Cashier Side
* Login Side
* Print the products inventory. (Using Crystal Report 2010)
* Add, Change, & Delete the following
  * User Accoun for Admin and Cashier
  * Products Enlisted to your inventory
